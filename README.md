# Janvi's Cybersecurity Portfolio & Blog 🛡️💻

Welcome! I am a Computer Science undergraduate with a strong interest in cybersecurity, ethical hacking, and secure system design. This repository contains my personal portfolio website and blog, where I document my learning journey, write exploitation reports, and share hands-on security testing projects.

## 🚀 About Me

I enjoy understanding how websites, networks, and systems work, identifying security vulnerabilities, and learning how to protect applications from different types of cyber threats. My primary areas of interest include:
- Network Scanning & Penetration Testing
- Web Application Security
- Vulnerability Assessment & Mitigation
- Secure Coding Practices

### 🛠️ Core Tools & Technologies
- **Scanning & Assessment:** Nmap, Nessus, Burp Suite, Metasploit
- **Languages & Frameworks:** Astro, TypeScript, TailwindCSS
- **Operating Systems:** Linux (Kali, Parrot), Windows

## 📂 Project Structure

This portfolio is built with [Astro](https://astro.build/) using a terminal-inspired theme:
- `src/content/posts/` — Cybersecurity blog posts, exploit reports, and writeups
- `src/pages/` — Main pages (Home, About, Archive)
- `src/site.config.ts` — Site-wide configuration and metadata

## 💻 Local Development

To run this project locally, clone the repository and run the following commands:

```bash
# Install dependencies
npm install

# Start the local development server
npm run dev

# Build for production
npm run build
```
